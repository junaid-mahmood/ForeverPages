# ForeverPages

ForeverPages is a web-based application that allows users to upload images or text, organize them in boxes, and search for specific contentâ€”essentially a dynamic, interactive bulletin board for the internet. - Create and manage custom pages with text and image uploads

- Responsive design for both mobile and desktop
- Easy setup and deployment
